# UK Flood Risk Insurance Exposure Analysis

## A catastrophe risk and exposure management project demonstrating flood hazard analysis, exposure aggregation and simplified catastrophe loss modelling using Python and SQL.

---

### Overview

This project was undertaken to develop practical analytical skills relevant to catastrophe risk and exposure management graduate roles.

Using publicly available UK postcode flood risk data, the project demonstrates a simplified catastrophe risk workflow including hazard analysis, exposure aggregation, SQLite database querying, concentration analysis and loss simulation.

Whilst simplified, the project reflects many of the analytical concepts used within catastrophe modelling and exposure management teams across the insurance and reinsurance industry.

### Analytical Techniques Demonstrated

- Flood hazard analysis
- Exposure aggregation
- Python (Pandas, NumPy, Matplotlib)
- SQL database querying (SQLite)
- Exploratory data analysis
- Statistical visualisation

### Key Features

- Analysis of UK postcode flood hazard classifications.
- SQL integration using SQLite for querying and aggregating data.
- Visualisation of flood risk distributions.
- Exposure aggregation by flood risk category.
- Simplified catastrophe loss simulation.
- Conceptual Average Annual Loss (AAL).
- Conceptual Probable Maximum Loss (PML).
